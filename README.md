# 集美大学 2026 数字逻辑 课程设计

## 进行时间
2026-06-29 - 2026-07-01

## 核心目标
通过 Logisim-evolution 实现能够识别单个正立手写数字的神经网络，权重使用 Pytorch 编写训练代码，基于 MNIST 数据集训练得到

## 路线图
- [ ] 在 Python 基于 MNIST 数据集，进行训练和推理
  - 由于要通过**多层**感知机实现，要将 MNIST 训练数据缩放为`8px*8px`进行训练
  - 目前计划总计3层2组权重，参数量为 64->16->10，全连接，以分类概率输出
- [ ] 将权重量化为 INT8，在 Python 显式实现矩阵运算，进行推理测试与评估
- [ ] 参考显式实现的矩阵运算器，在 Logisim-evolution 实现单组矩阵乘法器
- [ ] 设计状态机，实现权重动态加载，复用计算模块
  - 为了可控的交付周期，网络结构直接写在电路和“常量”中，不从外部加载（但相应位置会有说明）
